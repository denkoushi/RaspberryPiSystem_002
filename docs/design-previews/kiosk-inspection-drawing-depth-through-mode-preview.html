v2 合意ポイント（右ペイン）
  • 十字キーは ↑↓←→ の1行（現行コードの flex 配置に合わせる。2×2 は使わない）。
  • 「補足」見出し削除。面 / 呼び径 / 直接入力の3列はそのまま。
  • 「深さモード」見出しは出さない。深さ/ネジ穴深さのときだけ「測定｜通し」をタイトルなしで表示。
  • 基準値は ラベルと入力を同一行。入力幅は狭め（≈7.5rem）。OCR候補は入力の下。
  • 削除ボタン: 高さ約半分 · 文言 一点削除 / 全削除。
  • 通し時は基準値・公差をロック。null クリアだけでは判定が壊れるため明示モード。
