検査図面 指差し記号 — 実機レイアウト準拠プレビュー

対象は作成/改版画面（右ペイン 17rem・1280×720 想定）。 現行の設定パネルはすでに縦が詰まっている（ナッジ・名称・通し・補足・基準値・OCR・公差・削除・一覧）。 指差しは右ペインを膨らませず、キャンバス上で置く前提。

実装前・静的 本番 aside = 17rem 右ペイン増設は一覧を押し出す 大きな設定ブロック追加は NG

余白方針: 指差しの本体はキャンバス（引出線＋同じ番号バッジ）。 右ペインは 1行ステータス（あり/なし + 削除）だけ。 配置モードはヘッダーの「点を配置」と同列の 「指差し」モードボタンで切替（新パネルを増やさない）。 さらに縦を稼ぐため、ナッジ矢印は高さ半分、名称ラベル＋ドロップダウンは1行（ドロップダウンは右ペイン内に収まる幅）。

操作フロー（推奨）
  • 測定点（丸数字）を選択する
  • ヘッダーで「指差し」モードを ON
  • 図面上の穴/ネジ穴をタップ → 先端を置く（線は自動で選択中 No. へ）
  • 先端付近に同じ番号バッジを表示（連携が目で分かる）
  • 不要なら右ペイン1行の「削除」、または先端をドラッグし直し
  • 全項目必須にしない。一覧は「指」の小タグだけで有無を示す
右ペインの縦圧縮（今回反映）
  • ↑↓←→ ナッジ: 高さ 半分（約 1.375rem / 現行 min-h-11 の半分）
  • 名称: ラベルとドロップダウンを1行。select は flex:1; min-width:0 で 17rem 内に収める（はみ出しなし・長い名称は省略）
データ（実装時の示唆）
  • 既存の丸数字中心 xRatio/yRatio はそのまま
  • 任意フィールドとして tip 側 ratio（例: calloutTipXRatio/YRatio）を測定項目に紐づけ
  • 描画は InspectionDrawingCanvas 上の SVG レイヤ（右ペインに図形UIを置かない）
