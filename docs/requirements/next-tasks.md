**要件定義の該当箇所**:

**NFR-001: パフォーマンス** ([要件定義 79-87行目](./system-requirements.md#nfr-001-パフォーマンス))
```
### NFR-001: パフォーマンス
- **説明**: レスポンス時間が許容範囲内であること
- **詳細**:
  - APIレスポンス時間: 1秒以内（通常時）
  - ページ読み込み時間: 3秒以内
- **実装状況**: ✅ 実装済み（実機検証未実施）
```

**NFR-004: 保守性** ([要件定義 110-122行目](./system-requirements.md#nfr-004-保守性))
```
### NFR-004: 保守性

- **詳細**:
  - バックアップ・リストア機能 ✅ 実装済み（実機検証未実施）

- **実装状況**: ✅ 実装済み（一部実機検証未実施）
```
