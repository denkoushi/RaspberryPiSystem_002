## 優先度の高いタスク

### 1. Milestone 5: 実機検証フェーズ（最優先）

**優先度: 🔴 最高**

**理由**: システムの信頼性と実用性を確認するため、最も重要。

**残タスク**:
- [ ] Validation 6: オフライン耐性
  - NFCエージェントがオフライン状態で動作するか確認
  - SQLiteキューにイベントが保存されることを確認
  - オンライン復帰後にキューがフラッシュされることを確認
- [ ] Validation 7: USB一括登録
  - USBメモリからのCSVファイル読み込みを確認
  - 従業員・アイテムの一括登録が正常に動作することを確認
  - エラーハンドリング（不正なCSV形式など）を確認

**推定工数**: 2-3日

**次のアクション**: Validation 6（オフライン耐性）とValidation 7（USB一括登録）を実施。
