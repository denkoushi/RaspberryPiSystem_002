### 📋 ステップ2: 認証トークンの取得

認証トークンを取得する方法は2つあります：

#### 方法1: ブラウザの開発者ツールを使用（推奨）

1. **開発者ツールを開く**
   - 管理画面にログインした状態で、ブラウザの開発者ツールを開く（F12キー）
   - 「Application」タブ（Chrome）または「Storage」タブ（Firefox）を開く

2. **トークンを確認**
   - 左側のメニューから「Local Storage」または「Session Storage」を選択
   - `https://100.106.158.2`をクリック
   - `token`というキーを探す
   - 値をコピー（これが認証トークンです）

#### 方法2: ブラウザのコンソールを使用

1. **コンソールを開く**
   - 開発者ツールの「Console」タブを開く

2. **以下のコマンドを実行**
   ```javascript
   // Local Storageから取得を試みる
   localStorage.getItem('token') || sessionStorage.getItem('token')
   ```

3. **表示されたトークンをコピー**

#### 方法3: Networkタブから確認

1. **Networkタブを開く**
   - 開発者ツールの「Network」タブを開く
   - ページをリロード（F5キー）

2. **APIリクエストを確認**
   - `/api/`で始まるリクエストを探す
   - リクエストをクリックして「Headers」タブを開く
   - 「Request Headers」セクションで`Authorization: Bearer <トークン>`を確認
   - `<トークン>`の部分をコピー

### ⚠️ 注意事項

- トークンは機密情報です。他人に共有しないでください
- トークンには有効期限があります。期限切れの場合は再ログインが必要です
- トークンは`Bearer `の後の部分のみをコピーしてください（`Bearer `は含めない）
